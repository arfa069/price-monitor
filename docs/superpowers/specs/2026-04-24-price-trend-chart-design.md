# 价格趋势图功能设计

## 概述

在商品管理页面为每个商品添加价格趋势图查看功能。用户点击「查看趋势」按钮后，通过弹窗展示该商品的历史价格走势。

## 用户场景

用户想了解某个商品的价格变化趋势，判断是否值得入手或是否应该设置降价提醒。

## 功能详情

### 入口
- 在商品表格每行操作列添加「查看趋势」按钮
- 使用 LineChart 图标（来自 @ant-design/icons 或 lucide-react）
- 按钮位于「编辑」和「删除」之间

### 弹窗结构

```
┌────────────────────────────────────────────────────────────┐
│  [商品标题] 价格趋势                                  [X] │
├────────────────────────────────────────────────────────────┤
│  [7天] [30天] [90天] [全部]           ← 时间范围切换按钮组 │
├────────────────────────────────────────────────────────────┤
│  📉 最低价: ¥XX    📈 最高价: ¥XX                         │
│  💰 当前价: ¥XX    ⬇️ 降价次数: X次                       │
├────────────────────────────────────────────────────────────┤
│                                                              │
│           📈 价格折线图 (Recharts LineChart)                │
│           X轴: 日期, Y轴: 价格                             │
│           数据点有 tooltip 显示具体价格和日期               │
│                                                              │
├────────────────────────────────────────────────────────────┤
│  最近价格记录表格                                           │
│  列: 时间 | 价格                                            │
│  按时间倒序排列，显示最近 limit 条记录                      │
└────────────────────────────────────────────────────────────┘
```

### 时间范围
- 选项: 7天 / 30天 / 90天 / 全部
- 默认选中: 30天
- 切换时调用 `/products/{id}/history?days=X`

### 统计卡片
| 指标 | 说明 |
|------|------|
| 最低价 | 时间范围内最低价格 |
| 最高价 | 时间范围内最高价格 |
| 当前价 | 最新一条记录的价格 |
| 降价次数 | 时间范围内价格下降的次数 |

### 降价次数计算逻辑
遍历价格记录，当 `price[i] < price[i-1]` 时计数 +1。

### 空状态处理
- 无数据时显示「暂无价格记录」
- 图表区域显示空状态文案
- 统计卡片显示「-」

## 技术实现

### 依赖
- **Recharts**: 用于绘制折线图
  - 已在 frontend/package.json 中，需确认是否已安装
  - 如果未安装，执行 `cd frontend && npm install recharts`

### 新增文件
- `frontend/src/components/PriceTrendModal.tsx` — 趋势图弹窗组件

### 修改文件
- `frontend/src/pages/ProductsPage.tsx` — 添加「查看趋势」按钮和弹窗

### API 调用
```typescript
// 已存在的 API
products.history(id: number, days?: number, limit?: number): Promise<PriceHistoryRecord[]>
```

### Props 接口
```typescript
interface PriceTrendModalProps {
  open: boolean
  product: Product
  onCancel: () => void
}
```

## 组件状态

| 状态 | 处理 |
|------|------|
| loading | 显示 Skeleton |
| error | 显示错误提示，allowRetry |
| empty | 显示空状态 |
| success | 正常展示图表、统计、表格 |

## 图表配置

- 折线颜色: `#2563eb` (与主题 primary 色一致)
- X轴: 时间格式化显示 (如 "04-01")
- Y轴: 价格，格式化显示 ¥ 符号
- Tooltip: 显示完整日期时间 + 价格
- 无数据点时曲线断开
