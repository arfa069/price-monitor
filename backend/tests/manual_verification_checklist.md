# Phase C 手动验收清单

> 需要浏览器测试的用例。pytest 覆盖了 API 层，这些用例需要真实 UI 验证。

## 前置条件

1. 启动后端：`uvicorn app.main:app --reload`（不推荐用 `--reload`，Windows 上有 Playwright subprocess 问题）
2. 启动前端：`cd frontend && npm run dev`
3. 打开浏览器：`http://localhost:3000`
